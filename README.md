# SavingState-hw
---
Завдання 1
-
Створіть MVC програму, яка буде підраховувати кількість користувачів онлайн. Кількість користувачів повинна відображатися на сторінці програми. Щоб перевірити роботу програми, запустіть її в кілька браузерах. Якщо ви відкриєте програму в трьох браузерах, кількість користувачів онлайн має дорівнювати трьом.

На сервері отримана інформація повинна бути записана в Cookies із встановленням дати старіння, що дорівнює тій, яка була встановлена ??в другому полі введення у формі. Зробіть сторінку, яка буде використовуватися для перевірки наявності значення Cookies.
